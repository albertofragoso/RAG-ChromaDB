# RAG ChromaDB

Sistema RAG (Retrieval-Augmented Generation) utilizando ChromaDB y LangChain para el proyecto Café Aurora.

## Características
- Generación de respuestas basadas en documentos locales.
- API construida con FastAPI.
- Almacenamiento vectorial persistente con ChromaDB.
- Integración con modelos de HuggingFace y OpenAI.
